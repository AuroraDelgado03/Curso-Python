# Módulo 2: Avanzado

## Archivos del Proyecto

### `reto1_POO.py`
Crear una clase llamada **Componente**, que funcione como ficha técnica de cualquier pieza en una fábrica.  
Cada objeto generado debe:
- Inicializarse con datos básicos del componente  
- Contar con su **propio historial de revisiones**  
- Incluir métodos para agregar y consultar revisiones  

---

### `reto2_POO.py`
Función que recibe una cadena de texto (con o sin puntuación) y devuelve un **diccionario de frecuencia de palabras**.  
Características clave:
- Manejar mayúsculas y minúsculas (convertir todo a minúsculas)  
- Ignorar signos de puntuación  
- Separar palabras correctamente  
- Contabilizar cuántas veces aparece cada palabra  

---

### `reto3_POO.py`
Sistema de clases utilizando **Herencia** para modelar una flota de vehículos.  
Debe incluir:
- Clase padre: `Vehiculo`  
- Clases hijas: `Coche` y `Camion`  
- Atributos y métodos compartidos en la clase padre
