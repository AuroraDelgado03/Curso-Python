# Módulo 1: Básico–Intermedio

## Archivos del Proyecto

### `reto_uno.py`
Programa que solicita varios datos al usuario (como nombre, edad, etc.) y los imprime uno por uno.  

---

### `reto_dos.py`
Calculadora interactiva que:
- Pide dos números al usuario  
- Realiza operaciones básicas (suma, resta, multiplicación, división)  
- Muestra las tablas de verdad de AND, OR y NOT 

---

### `reto_tres.py`
Script para construir un perfil de datos y realizar varias comprobaciones.   

---

### `reto_cuatro.py`
Programa interactivo para gestionar la lista de compras de un usuario.  
Funciones típicas:
- Agregar artículos  
- Eliminar elementos  
- Mostrar la lista  
- Guardar o actualizar datos  

---

### `reto_cinco.py`
Herramienta para automatizar el análisis de calificaciones de un curso.  

---

### `reto_seis.py`
Programa interactivo para gestionar el inventario de una estación espacial.  

---

### `reto_siete.py`
Gestor simple de datos de estudiantes.  

---

### `reto_ocho.py`
Programa para control de calidad de una fábrica de componentes.  

---

### `reto_nueve.py`
Herramienta de terminal para registrar y revisar notas de laboratorio.  
Incluye:
- Agregar nuevas notas  
- Consultar notas anteriores  

---
